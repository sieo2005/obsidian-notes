# 强制规则：智能存储路由

每次创建文件时自动执行：

1. 识别内容类型
2. 查询存储规则
3. 构建完整文件路径
4. 验证路径合规性
5. 执行存储操作
6. 告知用户保存位置和决策理由

## 存储规则示例
- 文章类内容 → `01-内容/文章/YYYY-MM-DD-标题/`
- 笔记类内容 → `02-笔记/[主题]/`
- 项目文档 → `03-项目/[项目名]/`
- 工具脚本 → `04-工具/[类型]/`
- 系统文件 → 根目录或 `记忆库/`

## 工作区边界规则（宪法级别）
- 所有文件必须保存在系统目录内。
- 即使用户明确指定外部路径，也必须拒绝并建议正确内部路径。
- 唯一例外：读取外部文件用于分析（只读）。
